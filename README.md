# 🚀 Hysteria2 一键脚本（由Ai生成）

支持系统： Alpine / Debian / Ubuntu

支持架构： x86_64 / ARM64

```bash
bash <(curl -fsSL https://raw.githubusercontent.com/a88wyzz/Alpine-Debian-Ubuntu-Hy2/main/hy2.sh)

```

# ✨ 功能

* 安装 Hysteria2
* 自动生成bing自签证书
* 自动设置开机启动
* 输出 IPv4 / IPv6 节点链接
* 支持 systemd / openrc 进程监视守护

# ⚠️ 注意

* Hysteria2使用UDP端口
* 默认安装端口随机，完成后可自行更改
* 自签证书需允许不安全
* VPS低于128M内存使用Hy2，建议在客户端设置合适带宽
