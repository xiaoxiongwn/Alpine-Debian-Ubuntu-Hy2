# Juicity 一键脚本（由Ai生成）

支持系统：

* Alpine / Debian / Ubuntu

支持架构：

* x86_64 / ARM64

# 🚀 VPS运行以下脚本

```bash
bash <(curl -fsSL https://raw.githubusercontent.com/a88wyzz/Alpine-Debian-Ubuntu-Hy2/main/Juicity/juicity.sh)

```

# ✨ 功能

* 安装 Juicity
* 自动生成bing自签证书
* 自动设置开机启动
* 输出 IPv4 / IPv6 节点链接
* 支持 systemd / openrc 进程监视守护

# ⚠️ 注意

* 自签证书需允许不安全
