# TUIC 一键脚本（由Ai生成）

支持系统：

* Alpine
* Debian / Ubuntu

支持架构：

* x86_64
* ARM64

---

## 🚀 一键安装

```bash
bash <(curl -fsSL https://raw.githubusercontent.com/a88wyzz/Alpine-Debian-Ubuntu-Hy2/main/Tuic/tuic.sh)
```

---


## ✨ 功能

* 安装 TUIC
* 自动生bing自签证书
* 自动开机启动
* 输出 v2rayN 的 IPv4 / IPv6 链接
* 支持 systemd / openrc 进程监视守护

---

## ⚠️ 注意

* 默认端口随机，可自行更改
* 自签证书需允许不安全
